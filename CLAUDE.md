# SAP ECC 6.0 Knowledge Base

A structured reference for SAP ECC 6.0 (Enhancement Packs 0-8), designed as context for Claude Code sessions.

## What This Is

Curated SAP knowledge organized by module — transaction codes, SPRO configuration paths, process flows, master data, integration points, and solution patterns. Covers the four core logistics and finance modules: MM, SD, FI, CO.

## How to Use

1. Start with `.claude/rules/sap-routing.md` to identify the right module
2. Navigate to `modules/{module}/CLAUDE.md` for module orientation
3. Read specific content files as needed (tcodes, config, processes, etc.)

Additional content:
- `cross-module/` — End-to-end processes (Procure-to-Pay, Order-to-Cash, Record-to-Report)
- `reference/` — Lookup tables (movement types, document types, posting keys)

## Scope

This knowledge base covers **ECC 6.0 only**. Where S/4HANA differs, differences are noted for disambiguation — but S/4HANA is never the primary reference. Always check file frontmatter for confidence level and verification date.
