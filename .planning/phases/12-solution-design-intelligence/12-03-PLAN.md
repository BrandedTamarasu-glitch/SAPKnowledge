**Section 1: ## Month-End Close Checklist**

Use markdown checkbox format per locked decision: `- [ ] Step description (T-CODE) -- brief explanation`

Organize by business day timing per CONTEXT.md locked decision:

### Day -5 to Day -3 (Pre-Close Preparation)
- Verify all goods receipts posted (MIGO, ME2M for PO list)
- Verify all goods issues and transfers posted (MIGO)
- Confirm vendor invoices posted or accruals planned (MIRO, FBS1)
- Check for negative stock (MMBE)
- Verify all billing documents created (VF04 billing due list)
- Verify all SD deliveries have PGI posted (VL06G)

### Day 1 (MM Period-End)
- Run MMPV -- close MM posting period
- Run F.13 -- GR/IR automatic clearing
- Run MR11 -- GR/IR maintenance for one-sided items
- If Material Ledger active: run CKMLCP -- actual costing close

### Day 1-2 (FI Period-End)
- Run AFAB -- depreciation for the period
- Post period-end accruals (FBS1) -- reversing accruals for unrecorded expenses
- Run F.05 (or FAGL_FC_VAL) -- foreign currency revaluation
- Clear open items: F-32 (AR), F-44 (AP)
- Verify GL balances: FAGLB03

### Day 2-3 (CO Period-End)
- Run KB61 -- repost CO line item mis-assignments
- Run KGI2 -- calculate actual overhead
- Run KSU5 -- assessment cycles (TEST mode first, then LIVE)
- Run KSV5 -- distribution cycles (TEST mode first, then LIVE)
- Run KO88 -- settle internal orders (TEST mode first, then LIVE)
- Run CO88 -- settle production orders (if PP active)
- Run KSII -- calculate actual activity prices
- Run KALC -- reconciliation posting (if multi-CC controlling area)
- Lock CO period: OKP1

### Day 3-4 (Verification and Reporting)
- Run S_ALR_87013611 -- cost center plan/actual comparison
- Run KSB1 -- verify catch-all cost centers
- Run KE5Z vs FAGLB03 -- PCA-FI reconciliation
- Run MB5S -- verify GR/IR clearing account
- Distribute management reports
- Lock FI posting periods: OB52

**Section 2: ## Year-End Close Checklist**

Brief intro: year-end = complete month-end checklist above for period 12, THEN execute these additional steps.

Year-end-only steps (per RESEARCH.md):
- Balance carryforward GL: FAGLGVTR (or F.16 for classic GL)
- Balance carryforward AR: F.07
- Balance carryforward AP: F.16
- Asset fiscal year close: AJAB (after depreciation fully posted; CRITICAL: AJAB is irreversible in many configurations)
- CO fiscal year close: OKP1 lock all periods for the closed year
- Open new FI period for new fiscal year: OB52
- Open new MM period for new fiscal year: MMPV
- Copy CO planning version if needed: KP97/KP98

Include S/4HANA Differences section at end.
